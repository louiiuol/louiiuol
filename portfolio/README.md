## Available Scripts

In the project directory, you can run:

- `yarn start`: launches development server
- `yarn test`: launches written tests
- `yarn build`: builds the app for production to the `build` folder.
- `yarn deploy`: build and deploy project to production branch & Github Pages
- `yarn eject`: If you aren‚Äôt satisfied with the build tool and configuration choices, you can `eject` to remove the single build dependency.
